# AI Chat Exporter Browser Extensions

Export conversations from ChatGPT, Claude, Gemini, and other AI platforms to Markdown, PDF, Word, and more.

---

## 1. AI Chat to PDF `v3.5.2`

**Purpose:** Instantly export Gemini chats to PDF with smart conversion.

| | |
|---|---|
| **Chrome/Brave** | [AI Chat to PDF on Chrome Web Store](https://chromewebstore.google.com/detail/ai-chat-to-pdf/nifkaldinolmacagmiddfpnfaecfflbn) |
| **Firefox** | Not officially listed; use on a Chromium-based browser |

**Key Features:**
- Focused on Gemini chat export
- Smart PDF conversion for perfect formatting results
- Fast one-click export

## 2. AI Exporter — Save ChatGPT, Gemini to PDF… `v4.2.2`

**Purpose:** Multi-platform AI chat exporter supporting ChatGPT, Claude, Gemini, and NotebookLM.

| | |
|---|---|
| **Chrome/Brave** | [AI Exporter on Chrome Web Store](https://chromewebstore.google.com/detail/ai-exporter-save-chatgpt/kagjkiecagemklhmhkabbalfpbianbe) |
| **Firefox** | Not officially listed; use on a Chromium-based browser |

**Key Features:**
- Exports to **PDF, Word (.docx), Markdown (.md), and .txt**
- Works with ChatGPT, **Claude**, **Gemini**, and NotebookLM
- Supports content selection (export only part of a conversation)
- Notion sync support
- Download directly from the chat interface
