# Network Speed in Panel — Minimon Applet (COSMIC / Pop!_OS)

A DU Meter equivalent for Pop!_OS's COSMIC desktop. Minimon is a native COSMIC
panel applet (Rust + iced, Wayland-native) that shows live CPU, Memory,
Network, Disk, and GPU usage directly in the panel or dock.

Repo: https://github.com/cosmic-utils/minimon-applet

---

## Option A — Install via COSMIC Store (recommended)

1. Open the **COSMIC Store** app.
2. Search for `Minimon`.
3. It should appear under the **COSMIC Applets** category. Install it.

If it doesn't show up, add the COSMIC flatpak remote first, then search again:

```bash
flatpak remote-add --if-not-exists --user cosmic https://apt.pop-os.org/cosmic/cosmic.flatpakrepo
```

---

## Option B — Install via `.deb`

1. Download the latest `.deb` from the
   [releases page](https://github.com/cosmic-utils/minimon-applet/releases).
2. Install it:

```bash
sudo dpkg -i cosmic-applet-minimon_<version>_amd64.deb
```

---

## Option C — Build from source

```bash
sudo apt install just libxkbcommon-dev
git clone https://github.com/cosmic-utils/minimon-applet.git
cd minimon-applet
just build-release
just install
```

Generate a `.deb`/`.rpm` instead, if preferred:

```bash
just build-release
just deb   # or: just rpm
```

---

## Enable it on the Panel

1. Open **COSMIC Settings** → **Desktop** → **Panel** (or **Dock**).
2. Find **Minimon** in the applet list and enable it.
3. Click into its settings to choose what's shown — network up/down speed,
   CPU %, RAM, disk, GPU — and toggle each on/off.

---

## Notes

- Minimon only works as an applet *inside* COSMIC — it won't run as a
  standalone tray app on other desktop environments.
- It runs as its own self-contained `libcosmic` process (no X11/GTK tray
  hacks), so it behaves correctly under Wayland.
