C++ Tuple Usage in Minimum Spanning Tree

✓

Declaration

#include <vector>
#include <tuple>
using namespace std;

// Format: {source, destination, weight}
vector<tuple<int, int, int>> MST;
Pro Tip

Use descriptive type aliases to improve readability:

using Edge = tuple<int, int, int>;
vector<Edge> MST;
✓

Insertion

You can insert using make_tuple or brace initialization:

// Method 1: Using make_tuple (works in all C++ versions)
MST.push_back(make_tuple(u, v, weight));

// Method 2: Using brace initialization (C++11 and newer)
MST.push_back({u, v, weight});
Modern Approach

With C++17, you can use emplace_back for better performance:

MST.emplace_back(u, v, weight);
✓

Accessing Elements

Use get<index> to access tuple elements:

// Traditional access method
int u = get<0>(MST[i]);
int v = get<1>(MST[i]);
int w = get<2>(MST[i]);
C++17 Feature

Use structured bindings for cleaner code:

// Modern approach with structured bindings
auto [u, v, w] = MST[i];
✓

Iterating Through MST

Iterate through all edges in the MST:

// Using traditional for loop
for (int i = 0; i < MST.size(); ++i) {
    int u = get<0>(MST[i]);
    int v = get<1>(MST[i]);
    int w = get<2>(MST[i]);
    cout << u << " -- " << v << " : " << w << endl;
}

// Using range-based for loop with structured bindings (C++17)
for (const auto& [u, v, w] : MST) {
    cout << u << " -- " << v << " : " << w << endl;
}
