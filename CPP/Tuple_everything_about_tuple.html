✅Declaration: 
#include <vector>
#include <tuple>
using namespace std;

vector<tuple<int, int, int>> MST;
✅Insertion:

You can insert using make_tuple or brace initialization:

MST.push_back(make_tuple(u, v, weight));  // preferred
MST.push_back({u, v, weight});            // also works (C++11+)
✅Accessing elements:

Use get<index> to access tuple elements:

int u = get<0>(MST[i]);
int v = get<1>(MST[i]);
int w = get<2>(MST[i]);

Or use structured bindings (C++17+):

auto [u, v, w] = MST[i];
