Project Specification: WayClip — Wayland Clipboard Manager

Version: 1.2
Target OS: Pop!_OS / Ubuntu 24.04 LTS (Wayland + GNOME)
Author: Riyad
Status: Planning


1. Problem Statement

On Wayland + GNOME, no existing clipboard manager supports auto-paste on item selection. This is a fundamental Wayland security restriction — apps cannot control each other's input at the userspace level. Existing tools like CopyQ and Clipboard Indicator either crash, fail silently, or require manual Ctrl+V after selection.

WayClip solves this by:


Using wl-clipboard (native Wayland clipboard tool) for reading/writing
Using ydotool (kernel-level input simulator via /dev/uinput) for auto-paste
Running as a lightweight background daemon
Showing a fast popup UI bound to a single key (Super+V)

4. Storage Design

No database. Two plain text files + one image folder.

~/.local/share/wayclip/
├── history.txt       ← last 20 recent items (text + image paths)
├── pinned.txt        ← pinned items (text + image paths), never auto-deleted
└── images/           ← PNG files for image clipboard items
    ├── img_1749.png
    ├── img_1750.png
    └── ...

Line format in history.txt and pinned.txt

Every line starts with a type prefix:

TEXT:hello world
TEXT:sudo apt install python3
IMAGE:/home/riyadunnabi/.local/share/wayclip/images/img_1749.png
TEXT:SELECT * FROM users
IMAGE:/home/riyadunnabi/.local/share/wayclip/images/img_1750.png


TEXT: → the rest of the line is the clipboard text (newlines escaped as \n)
IMAGE: → the rest of the line is the absolute path to a saved PNG file


Image cleanup rule

When an image item is dropped from history (21st item) or manually deleted, its PNG file in images/ is also deleted to avoid orphaned files.

Pinned image items keep their PNG file as long as they remain pinned.

Components

1. Daemon (wayclip-daemon)


Runs in background on login
Watches text clipboard: wl-paste --watch
Watches image clipboard: wl-paste --watch --type image/png
On text change → prepends TEXT:... to history.txt
On image change → saves PNG to images/, prepends IMAGE:path to history.txt
Deduplicates consecutive identical items
Trims history.txt to max 20 lines, deletes orphaned image files


2. Popup UI (wayclip-ui)


GTK4 window (frameless, floating)
Opens on Super+V keypress
Closes on Escape or clicking outside
Search bar at top (filters text items in real-time; images always shown)
Recent items at top
Pinned items at bottom, separated by a divider
Text items → show text content
Image items → show thumbnail preview
Left-click → paste + close
Right-click → context menu (Pin / Unpin / Delete)


3. Autostart


Daemon registered as a systemd user service
Starts automatically on login

Item Card Rules


Every item card is fixed height: 64px regardless of content type or length
Text items show max 3 lines, truncated with ... if longer
Single line items (short text, URLs) show centered in the card
Image items show a thumbnail scaled to fit the 64px height
No card ever grows taller than 64px — the list stays uniform and clean
Full content is only seen after pasting


Interactions:


Left-click text item → wl-copy text + auto-paste + close
Left-click image item → wl-copy --type image/png image + auto-paste + close
Right-click any item → Pin / Unpin / Delete
Escape → close without pasting
Typing in search → filters text items in real-time


On item click (paste flow):


wl-copy the selected item (text or image)
Close the popup window
50ms delay
ydotool key ctrl+v

10. Daemon Logic (wayclip-daemon.py)

python# Pseudocode — runs two watchers in parallel

# Watcher 1: text
run: wl-paste --watch
on output:
    if content == last_text: skip
    if content is empty: skip
    if len(content) > 1MB: skip
    prepend "TEXT:" + escape(content) to history.txt
    trim_history()
    last_text = content

# Watcher 2: image
run: wl-paste --watch --type image/png
on output:
    save PNG bytes to images/img_<timestamp>.png
    prepend "IMAGE:" + filepath to history.txt
    trim_history()

def trim_history():
    lines = read history.txt
    if len(lines) > 20:
        dropped = lines[20:]
        for line in dropped:
            if line starts with "IMAGE:":
                delete the PNG file
        write lines[:20] to history.txt

Edge cases:


Empty clipboard → skip
Items > 1MB → skip
Daemon already running → check PID file, exit if duplicate
Text with newlines → escape as \n before storing
