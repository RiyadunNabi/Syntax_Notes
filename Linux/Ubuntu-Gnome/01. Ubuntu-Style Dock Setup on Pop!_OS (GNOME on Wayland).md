# Ubuntu-Style Dock Setup on Pop!_OS (GNOME on Wayland)

## Context
- OS: Pop!_OS 24.04 LTS
- Session: GNOME on Wayland
- Goal: Get Ubuntu-style left dock like the lab

---

## Step 1: Install GNOME Session
```bash
sudo apt install gnome-session
```
This adds GNOME session options to the login screen.

---

## Step 2: Install GNOME Tweaks
```bash
sudo apt install gnome-tweaks
```
Opens as **Tweaks** app. Go to **Window Titlebars** → enable **Minimize** and **Maximize** buttons.

---

## Step 3: Install Extension Manager + Ubuntu Dock
```bash
sudo apt install gnome-shell-extension-manager
sudo apt install gnome-shell-extension-ubuntu-dock
```

---

## Step 4: Enable the Dock

### Way 1 — Terminal
```bash
gnome-extensions enable ubuntu-dock@ubuntu.com
```

### Way 2 — Extension Manager UI
Open **Extension Manager** app → find **Ubuntu Dock** → toggle it ON.

Then **log out and log back in** (either way).

---

## Step 5: Move Dock to Left Side

### Way 1 — Terminal
```bash
gsettings set org.gnome.shell.extensions.dash-to-dock dock-position 'LEFT'
```

### Way 2 — Extension Manager UI
Open **Extension Manager** → find **Ubuntu Dock** → click the ⚙️ settings icon → set **Position** to **Left**.

---

## Step 6: Configure Auto-Hide

### Way 1 — Terminal
```bash
# Disable auto-hide (always visible)
gsettings set org.gnome.shell.extensions.dash-to-dock autohide false
gsettings set org.gnome.shell.extensions.dash-to-dock intellihide false

# Re-enable auto-hide
gsettings reset org.gnome.shell.extensions.dash-to-dock autohide
gsettings reset org.gnome.shell.extensions.dash-to-dock intellihide
```

### Way 2 — Extension Manager UI
Open **Extension Manager** → **Ubuntu Dock** → ⚙️ → toggle **Auto-hide** on/off visually.

---

## Notes
- `gnome-shell-extension-dashtodock` package doesn't exist on Pop!_OS — use `gnome-shell-extension-ubuntu-dock` instead
- Alt+F2 → `r` (GNOME shell restart) does NOT work on Wayland — log out/in instead
- If dock disappears after gsettings commands, reset with the reset commands in Step 6
