# Browser Extensions for Competitive Programming

Essential extensions for Chrome/Brave and Firefox to streamline your CP workflow.

---

## 1. CF Analytics

**Purpose:** Analyse and visualize Codeforces profiles — ratings, problem stats, contest history, and more.

| | |
|---|---|
| **Chrome/Brave** | [CF Analytics on Chrome Web Store](https://chromewebstore.google.com/detail/cf-analytics/hhljbjodjdbjggddjaidjajnlmaobcpo) |
| **Firefox** | Not officially listed; use the Chrome version via a Chromium-based browser |

**Key Features:**
- Visual rating graph and contest history
- Problem-solving statistics by tag and difficulty
- Compare profiles side by side

---

## 2. CPH Submit (Competitive Programming Helper)

**Purpose:** Automatically submit solutions to competitive programming judges directly from your local editor (VS Code).

| | |
|---|---|
| **Chrome/Brave** | [CPH Submit on Chrome Web Store](https://chromewebstore.google.com/detail/cph-submit/ekpinobooilkgpdbobcciehbhcmilkignc) |
| **Firefox** | [CPH Submit on Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/cph-submit/) |
| **VS Code Extension** | [Competitive Programming Helper (CPH)](https://marketplace.visualstudio.com/items?itemName=DivyanshuAgrawal.competitive-programming-helper) — required companion |

**Key Features:**
- One-click submit from VS Code to Codeforces, AtCoder, and more
- Works alongside the CPH VS Code extension
- Supports multiple languages

> **Note:** The browser extension alone is not enough — you also need the CPH VS Code extension installed.

---

## 3. Competitive Companion

**Purpose:** Parse problems and contests from competitive programming sites and send test cases to your local CP tool (CPH, Competitive Companion helper, etc.).

| | |
|---|---|
| **Chrome/Brave** | [Competitive Companion on Chrome Web Store](https://chromewebstore.google.com/detail/competitive-companion/cjnmckjndlpiamhfimnnjimnckgghkjbi) |
| **Firefox** | [Competitive Companion on Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/competitive-companion/) |

**Key Features:**
- Parses problems from 30+ sites: Codeforces, AtCoder, LeetCode, SPOJ, CodeChef, and more
- Sends test cases (input/output) directly to CPH or other local tools
- Works on both individual problems and full contest pages

**Supported Sites (selection):**
- Codeforces
- AtCoder
- LeetCode
- SPOJ
- CodeChef
- HackerRank
- DMOJ

---

## Recommended Setup

```
Competitive Companion  →  parses problems from browser
        ↓
CPH (VS Code)          →  receives test cases, runs locally
        ↓
CPH Submit             →  submits solution back to the judge
```

**Suggested workflow:**
1. Open a problem/contest in your browser.
2. Click **Competitive Companion** to parse and send test cases to CPH.
3. Write and test your solution locally in VS Code using CPH.
4. Click **CPH Submit** (or use the VS Code command) to submit directly.
5. Check your submission stats later with **CF Analytics**.
