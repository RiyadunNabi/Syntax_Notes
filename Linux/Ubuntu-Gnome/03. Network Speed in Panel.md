# Network Speed in Panel — GNOME on Wayland (Pop!_OS)
A Minimon equivalent for GNOME desktop. Shows live network upload/download
speed (and optionally CPU, RAM) directly in the top panel.

---

## Option A — Via Extension Manager (recommended)

1. Install Extension Manager if not already installed:
```bash
sudo apt install gnome-shell-extension-manager
```

2. Open **Extension Manager** app → click **Browse** tab.
3. Search for `Net Speed Simplified`.
4. Click **Install** → then toggle it **ON**.
5. Click the ⚙️ settings icon to choose what's shown (network speed, CPU, RAM, etc.).

---

## Option B — Via Browser (extensions.gnome.org)

1. Open Firefox/Brave and go to: https://extensions.gnome.org
2. Install the **GNOME Shell Integration** browser extension when prompted.
3. Also install the host connector:
```bash
sudo apt install chrome-gnome-shell
```
4. Search `Net Speed Simplified` on the site → toggle the switch to **ON** to install.

---

## Enable / Configure

- Open **Extension Manager** → find **Net Speed Simplified** → click ⚙️
- Toggle on/off: network up/down speed, CPU %, RAM usage
- Choose position in panel (left/center/right)

---

## Notes
- Unlike Minimon, this works on **GNOME only** (not COSMIC)
- Wayland-compatible — no X11 hacks needed
- If extension doesn't appear after install, log out and log back in
