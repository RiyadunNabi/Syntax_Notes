# Brave Browser Profile Shortcuts on Pop!_OS

## What we did

Created separate `.desktop` files so each Brave profile launches directly without the profile picker.

---

## Your profiles

| Profile | Directory | Shortcut file |
|---------|-----------|---------------|
| RiyadunNabi2 | `Profile 7` | `brave-profile7.desktop` |
| Prince | `Profile 8` | `brave-profile8.desktop` |

---

## Desktop file contents

### Profile 7 — RiyadunNabi2

File location: `~/.local/share/applications/brave-profile7.desktop`

```
[Desktop Entry]
Version=1.0
Name=Brave (RiyadunNabi2)
GenericName=Web Browser
Exec=/usr/bin/brave-browser --profile-directory="Profile 7" %U
Icon=/usr/share/icons/hicolor/256x256/apps/brave-browser.png
Terminal=false
Type=Application
Categories=Network;WebBrowser;
StartupWMClass=brave-browser
```

### Profile 8 — Prince

## How to launch

Since pinning to the dock shows a broken gear icon, use these methods instead:

### Method 1 — App launcher (recommended)

Press the **Super** key (or click the grid icon), search the profile name:

- `Brave (RiyadunNabi2)`
- `Brave (Prince)`

Click to launch. Each opens directly into its profile.

### Method 2 — Terminal

```bash
# Launch Profile 7
gtk-launch brave-profile7

# Launch Profile 8
gtk-launch brave-profile8
```

### Method 3 — Direct command

```bash
# Launch Profile 7
/usr/bin/brave-browser --profile-directory="Profile 7"

# Launch Profile 8
/usr/bin/brave-browser --profile-directory="Profile 8"
```

## If you ever need to recreate them

```bash
# Create the file
nano ~/.local/share/applications/brave-profileX.desktop

# Make executable
chmod +x ~/.local/share/applications/brave-profileX.desktop

# Mark as trusted
gio set ~/.local/share/applications/brave-profileX.desktop metadata::trusted true

# Refresh app database
update-desktop-database ~/.local/share/applications/
```

## To find a profile's directory name

Open that profile in Brave, go to `brave://version`, and check the **Profile Path** field. The last folder name (e.g. `Profile 7`) is what goes in `--profile-directory="..."`.

---

## To delete a shortcut

```bash
rm ~/.local/share/applications/brave-profileX.desktop
update-desktop-database ~/.local/share/applications/
```
